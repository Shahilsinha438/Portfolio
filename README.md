🆘 TROUBLESHOOTING QUICK LINKS
─────────────────────────────────

Issue: Form not submitting?
  → Check browser console (F12)
  → Verify firebase-config.js is correct
  → See: FIREBASE_SETUP_GUIDE.md → Troubleshooting

Issue: Email not received?
  → Check you enabled 2FA on Gmail
  → Check you used App Password (not regular password)
  → Check spam folder
  → See: QUICK_START.txt → TROUBLESHOOTING

Issue: Can't deploy to Firebase?
  → Run: firebase login
  → Run: firebase init
  → See: DEPLOYMENT_GUIDE.txt

Issue: Firebase errors in console?
  → Check CDN links are correct
  → Check firebase-config.js credentials
  → See: FIREBASE_SETUP_GUIDE.md
