🎯 WHAT YOU NEED TO DO
─────────────────────────────────────────────────────────────────────────────

STEP 1: Have Node.js Installed
       Download: https://nodejs.org (LTS version)
       
STEP 2: Get Your Firebase Project ID
       1. Go to: https://console.firebase.google.com
       2. Open your project
       3. Copy project ID (shown in top left)
       
STEP 3: Update .firebaserc
       Open the file, replace YOUR_PROJECT_ID with your actual ID
       
STEP 4: Update firebase-config.js
       1. Get credentials from Firebase Console → Project Settings
       2. Copy apiKey, projectId, authDomain, storageBucket, etc.
       3. Paste into firebase-config.js replacing placeholders
       
STEP 5: Open Command Prompt in Project Folder
       Right-click folder → "Open PowerShell here"
       
STEP 6: Install Firebase CLI (one-time)
       Run: npm install -g firebase-tools
       
STEP 7: Login to Firebase
       Run: firebase login
       (Browser opens for authentication)
       
STEP 8: Deploy!
       Run: firebase deploy --only hosting
       
STEP 9: Visit Your Live Site
       Copy URL from deployment output (https://your-project-id.web.app)

❓ COMMON QUESTIONS
─────────────────────────────────────────────────────────────────────────────

Q: Will my contact form work?
A: Yes! It submits data to Firebase Firestore. Make sure Firestore security
   rules are properly configured (see COMPLETE_DEPLOYMENT_GUIDE.md)

Q: Can I use a custom domain?
A: Yes! Buy a domain (~$1/year from Namecheap), then connect it in Firebase
   Console → Hosting → "Connect domain"

Q: How do I see submitted contact messages?
A: Firebase Console → Firestore Database → contacts collection

Q: Can I edit portfolio after deploying?
A: Yes! Edit files, run: firebase deploy --only hosting

Q: Is my data secure?
A: Yes! Firebase handles security, SSL encryption, and backups.

Q: How much does it cost?
A: Free! Firebase free tier includes:
   - 10 GB/month storage
   - 1 GB/month downloads
   - Unlimited hosting deployments
   - SSL certificate
