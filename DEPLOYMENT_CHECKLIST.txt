PHASE 4: FIRESTORE SETUP (IMPORTANT!)
───────────────────────────────────────────────────────────────────────────────

☐ Go to: https://console.firebase.google.com

☐ Select your project → Firestore Database

☐ Go to Rules tab

☐ Delete existing rules and paste:
   
   rules_version = '2';
   service cloud.firestore {
     match /databases/{database}/documents {
       match /contacts/{document=**} {
         allow read, write: if true;
       }
       match /blogs/{document=**} {
         allow read, write: if true;
       }
       match /projects/{document=**} {
         allow read, write: if true;
       }
     }
   }

☐ Click "Publish"

☐ Verify: Rules should show "1 rule"

TROUBLESHOOTING QUICK REFERENCE
───────────────────────────────────────────────────────────────────────────────

Problem: "Cannot find firebase-tools"
Solution: Run → npm install -g firebase-tools

Problem: "Invalid project ID"
Solution: Check .firebaserc file has correct ID
         Run → firebase projects:list

Problem: Contact form won't submit
Solution: 1. Check firebase-config.js has REAL credentials
         2. Check Firestore rules are published
         3. Open browser console (F12) for errors

Problem: Admin panel shows 404
Solution: Try accessing: https://your-project-id.web.app/admin.html

Problem: Can't login to Firebase
Solution: Run → firebase login --reauth
