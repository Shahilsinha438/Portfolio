🆘 TROUBLESHOOTING
────────────────────

Issue: Admin dashboard not loading
  → Check firebase-config.js has correct credentials
  → Open browser console (F12) for errors
  → Verify admin.html is in correct location

Issue: Blog posts not showing
  → Check Firestore has 'blog_posts' collection
  → Verify documents have required fields
  → Check browser console for Firebase errors

Issue: Analytics not tracking
  → Verify Google Analytics ID is correct
  → Check gtag() function is called
  → Look at Google Analytics dashboard

Issue: PWA not working offline
  → Clear browser cache
  → Re-register service worker
  → Check service-worker.js is in root
  → Verify manifest.json is linked

Issue: Performance issues
  → Run Lighthouse audit (F12 → Lighthouse)
  → Compress images
  → Minify CSS/JS
  → Enable caching headers


Got stuck?
  → Review IMPLEMENTATION_GUIDE.txt
  → Check ARCHITECTURE.md for system design
  → See ADVANCED_FEATURES_SETUP.txt for specific features
  → Review browser console for error messages
